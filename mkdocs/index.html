    mkdocs-material 的使用

    Note

    • 暂时先简要记录一下站点创建过程，实际使用过程中配置还不少，需要参考mkdocs-material的官方文档。
    • 参考文档：https://squidfunk.github.io/mkdocs-material/getting-started/

    构建步骤

    安装环境

    # 要求 Python 3.8及以上，且pip是可用的
# 下载虚拟环境工具
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple virtualenv

# 创建虚拟环境
cd /doc2023
virtualenv --python=python3 venv-mkdoc

# 激活虚拟环境
source venv-mkdoc/bin/activate

# 退出虚拟环境命令：deactivate

# 安装mkdocs-material
pip install mkdocs-material

    创建站点

    mkdir wait_gordon
cd wait_gordon
mkdocs new .

# 这将创建以下结构：
.
├─ docs/
│  └─ index.md
└─ mkdocs.yml

    在Python中，YAML（YAML Ain't Markup Language）是一种用于数据序列化的人类可读格式。它通常用于配置文件和在不同编程语言之间传递数据。YAML文件以.yml或.yaml为扩展名。 MkDocs是一个静态网站生成器，它可以将Markdown文档转换为HTML网站。为了简化MkDocs的配置，Material for MkDocs提供了一个名为schema.json的文件，该文件定义了mkdocs.yml文件的JSON模式。这样，当使用支持YAML模式验证的编辑器时，可以提供更好的语法高亮、自动补全和错误检查功能。

    Material for Mkdom为mkdocs.yml提供了自己的schema.json1 ，如果我们使用的是Visual Studio Code编辑器，则可以安装YAML插件，然后在设置中启用YAML模式验证。

    vscode 安装 YAML 扩展

    在用户或工作区的settings.json 中添加 yaml.schemas键下添加模式 我在此处选择工作区设置

    yaml配置内容 

    {
  "yaml.schemas": {
    "https://squidfunk.github.io/mkdocs-material/schema.json": "mkdocs.yml"
  },
  "yaml.customTags": [ 
    "!ENV scalar",
    "!ENV sequence",
    "!relative scalar",
    "tag:yaml.org,2002:python/name:material.extensions.emoji.to_svg",
    "tag:yaml.org,2002:python/name:material.extensions.emoji.twemoji",
    "tag:yaml.org,2002:python/name:pymdownx.superfences.fence_code_format"
  ]
}

    插件

    图片插件：MkDocs GLightbox

    支持图片打开和缩放

    1. 安装插件 pip install mkdocs-glightbox

    2.将 glightbox 添加到 mkdocs.yml 配置文件中 

    plugins:
 - glightbox

    1. 自定义插件配置 
      plugins:
  - glightbox:
      touchNavigation: true
      loop: false
      effect: zoom
      slide_effect: slide
      width: 100%
      height: auto
      zoomable: true
      draggable: true
      skip_classes:
        - custom-skip-class-name
      auto_caption: false
      caption_position: bottom
      background: white
      shadow: true
    Option Default Description
    touchNavigation true 启用或禁用触摸导航（滑动）
    loop false 循环幻灯片结束。
    effect zoom 灯箱效果 (zoom, fade, none)
    slide_effect slide 滑动效果. (slide, zoom, fade, none)
    width auto 内联元素和iframe的宽度。您可以使用任何单位，例如90%或100vw的全宽。
    height auto 内联元素和iframe的高度。您可以使用任何单位，例如90%，100vh或自动。
    zoomable true 启用或禁用可缩放图像。
    draggable true 启用或禁用鼠标拖动到上一张和下一张幻灯片。
    skip_classes [ ] 禁用特定自定义类名的图像的灯箱。
    auto_caption false 自动启用或禁用使用图片的alt作为字幕标题。
    caption_position bottom 默认字幕位置。 (bottom, top, left, right)
    background white lightbox图片的背景CSS。当图像为透明时，背景将显示。您可以使用任何CSS值作为背景，例如#74b9ff或Gainsboro或none。
    shadow true 启用或禁用灯箱图像的阴影。当背景为none时禁用它，以防止透明图像周围出现阴影。
